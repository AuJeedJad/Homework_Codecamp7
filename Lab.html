2. Differentiation
Tag Similarity Differences
hr & br Used for the same line break hr - Horizontal Line Break , br - Empty Line Break
ul & ol Used for dividing the bullet ul - Gives ordered list , ol - Gives unordered list
select & checkbox & radio button It is use for The "input tag" is the same They are different types of input
div & span Used for setting the style as well div is for text boxes in the line before or the next , span is for text boxes in the same line
script & style Defines the look of your page script - used for styling your page from Javascript , style - used for styling your page by css

3.Input Type
Input Type Usage
text It defines a one-line text input field
number Defines a numeric input field can also set restrictions on what numbers are accepted
checkbox Defines a checkbox let a user select
radio Defines a Radio buttons let a user select ONLY ONE of a limited number of choices
email Used for input fields that should contain an e-mail address
password Defines a password field. The characters in a password field are masked (shown as asterisks or circles)
color Used for input fields that should contain a color. a color picker can show up in the input field
date Used for input fields that should contain a date. a date picker can show up in the input field
time Show a Time Input Control. allows the user to select a time (no time zone)
date and time Local Date Field specifies a date and time input field, with no time zone
week Display a Week Input Control. allows the user to select a week and year
month Month Field allows the user to select a month and year
range Defines a control for entering a number whose exact value is not important can be displayed as a slider control
search Search Field is used for search fields (behaves like a regular text field
tel Used for input fields that should contain a telephone number
url Used for input fields that should contain a URL address
